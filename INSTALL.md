# Doc Nexus — Інструкція розгортання

## Вимоги

| | Версія |
|---|---|
| Node.js | v18+ (розроблено на v24) |
| npm | v9+ |
| Git | будь-яка |

---

## Веб-версія (браузер)

```bash
# 1. Встановити залежності
npm install

# 2. Запустити локально (розробка)
npm run dev
# → http://localhost:1420

# 3. Або зібрати для продакшн-деплою
npm run build
# → папка dist/ — статичні файли, кладуться на будь-який веб-сервер
```

---

## Десктоп-версія (Tauri)

Додатково потрібен **Rust**:

```bash
# Встановити Rust (https://rustup.rs)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# Потім
npm install
npm run tauri:dev    # розробка
npm run tauri:build  # збірка .dmg (Mac) або .msi (Windows)
```

---

## Перший запуск

1. Відкрити http://localhost:1420
2. В хедері → вставити **Anthropic API Key**
3. (Опціонально) вставити **Tavily API Key** для пошуку референсів
4. Ключі зберігаються в localStorage браузера

---

## Змінні середовища

Не потрібні — API ключі вводяться через UI і зберігаються в localStorage.

---

## Важливо

- `node_modules/` і `dist/` — не переносити, вони генеруються локально
- Запити до Claude API йдуть **напряму з браузера** — бекенд не потрібен
- При деплої на хостинг: просто покласти вміст `dist/` на статичний сервер (Nginx, GitHub Pages, Vercel тощо)
